# workout-processor
workout-processor is a program for organizing and processing workouts from a spreadsheet format
such as xlsx or csv into a more human-readable and presentable html file.

## Usage
Ensure your workout program is in the same directory you are running from and that it is one of the
following file types: .xlsx, .ods, .pdf, .html, .csv

Run the workout processor with the following command:
'python workout-processor.py your-workout-program.xlsx'
